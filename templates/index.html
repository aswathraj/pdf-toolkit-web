Python + HTML Document Suite

PDF Forge

Convert, merge, split, clean, and OCR files in one place. The app stores uploads on disk, removes the Flask upload cap, and processes documents locally.

No app-side file limit Runs locally Searchable OCR PDF

What’s included

  • JPG, Word, Excel, and PDF conversion flows
  • Merge and split PDF tools
  • Watermark cleanup with optional text matching
  • OCR via Tesseract for scans and images

“No limit” here means there is no artificial Flask upload cap in the app itself. Actual throughput still depends on machine storage, memory, and any reverse proxy you place in front of it.
