About The Project

PDF Forge was built to make document work simpler on the web and on desktop.

PDF Forge brings together conversion, PDF cleanup, merge and split workflows, and OCR in one place. It is designed to work as a public browser-based web app while still packaging cleanly as a desktop application.

{{ tool_count }} built-in tools Browser-ready deployment OCR-enabled workflows

Made by Aswath Raj

The app is built with Python, Flask, HTML, CSS, and a document-processing stack that covers images, PDFs, Word files, Excel files, and OCR for scanned documents.

When deployed publicly, files are processed on the server and old jobs are cleaned up automatically after about {{ job_retention_hours }} hours.

Capabilities

What this app covers

Document conversion

Convert JPG, PNG, and WEBP to PDF, resize images by pixels, set DPI, and move Word and Excel files in both directions.

PDF operations

Merge multiple PDFs, split by page or range, and clean watermarks or annotations.

OCR processing

Run Tesseract-powered OCR on scanned PDFs and images to generate searchable PDFs.
