# macOS Installer

Download the packaged desktop app here:

- [`PDFForge-macOS.dmg`](PDFForge-macOS.dmg)
- [`PDFForge-macOS.dmg.sha256`](PDFForge-macOS.dmg.sha256)

After opening the DMG:

1. Drag `PDF Forge.app` into `Applications`
2. Open `PDF Forge.app`
3. If macOS warns on first launch, right-click the app and choose `Open`
